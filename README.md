\# 📄 Resume Chatbot (RAG)



An AI-powered chatbot that answers questions about your resume using Retrieval-Augmented Generation (RAG).



\## 🚀 Features



\* Upload PDF resume

\* Ask questions about your experience

\* Context-based answers (no hallucination)

\* Fast retrieval using FAISS



\## 🛠️ Tech Stack



\* Python

\* Streamlit

\* LangChain

\* FAISS

\* HuggingFace Embeddings

\* Groq (LLaMA 3)



\## 📂 Project Structure



rag-resume-project/

│── app.py

│── requirements.txt

│── README.md



\## ⚙️ Setup Instructions



git clone https://github.com/Prashantpp6/rag-resume-project.git

cd rag-resume-project

pip install -r requirements.txt

streamlit run app.py



\## 🔑 Environment Variables



Create `.env` file:

GROQ\_API\_KEY=your\_api\_key\_here
